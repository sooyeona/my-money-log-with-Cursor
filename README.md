## 개발 노트
이 프로젝트는 **'바이브코딩'** 스타일을 지향하며, **Cursor AI**의 프롬프트 기능을 활용해 
복잡한 가계부 로직을 빠르게 구현하고 UI/UX 설계에 더 집중해 본 실험적인 프로젝트입니다.



##  my-money-log-with-Cursor



###  Project Overview
- **개발 동기:** Cursor AI의 기능을 활용해 빠르고 정확한 가계부 로직 구현
- **주요 기능:** 일별 소비 내역 기록, 일별/월별/연도별 소비 확인, 사용자추가, 지출 한도 설정, 다크모드 , jsonServer 저장
- **사용 기술:** React, TypeScript, emotion
